# DOM Wallet V3

DOM Wallet V3 is a new, independent, DOM-native wallet implementation.

## Objective

Build a robust wallet by combining validated DOM Wallet V1/V2 knowledge with mature engineering strategies studied from Epic Wallet, without copying Epic source code and without changing the philosophy or protocol rules of DOM.

## Sovereignty

DOM consensus, cryptography, privacy, chain identity, wire formats, fees, weights, coinbase rules, and economic policy are authoritative.

## Current phase

Specification and architecture foundation. Functional wallet implementation is not authorized yet.

## Completion rule

A feature is not complete merely because its happy path works. It requires explicit invariants, state transitions, atomic persistence, crash recovery, reorganization handling, and adversarial tests.

## Safety

Do not use this repository with mainnet, real funds, production seeds, or production wallet data.
